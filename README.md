A fun and interactive console-based number guessing game written in C#. The program generates a random secret number, and the player must guess it using logical hints provided by the system.

##  Features
* Randomized Logic: A new secret number is generated every time you play.
* Interactive Feedback: The game tells you whether your guess is "too high" or "too low".
* Attempt Tracking: Counts how many tries it took you to find the correct answer.
